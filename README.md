# Portfolio

My personal portfolio website built with Astro.

## Development

```sh
npm install
npm run dev
```

Open http://localhost:4321

## Build

```sh
npm run build
```

Output is in the `dist/` directory.

## Customize

- `src/pages/index.astro` - Profile name and bio
- `src/data/links.ts` - Social links
- `src/data/projects.ts` - Project list
- `src/styles/global.css` - Styles
